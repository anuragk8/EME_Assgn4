Human Anatomy Quiz Application

This interactive quiz project was developed through a structured process, focusing on user experience and functionality. The design phase involved planning the quiz structure, questions, and overall layout. The implementation utilized HTML for structure, CSS for styling, and JavaScript for interactive elements. Bootstrap was incorporated for responsive design and consistent styling across the quiz. The quiz features a clean interface with a navigation bar, a carousel for questions, and radio buttons for answer choices. JavaScript functions handle answer validation, providing immediate feedback to users with color-coded messages for correct, incorrect, or missing responses. To make the quiz accessible online, the project was deployed using GitHub Pages. This platform allows for easy updates and maintenance, ensuring that any improvements or changes can be quickly implemented and shared with users. The result is a user-friendly, interactive quiz that's easily accessible via a web browser, showcasing both the content knowledge and web development skills of its creator.

Drag-n-Drop Car Model Identification Quiz

The "Eclipse Publishing" logo design balances professionalism, creativity, and global influence. The central globe, crafted with smooth geometric lines using vector-based shapes, symbolizes the organization's far-reaching impact and the global spread of knowledge through publishing. This approach ensures clean, scalable lines that maintain quality at various sizes. The color palette of navy blue and gold evokes trust, sophistication, and creativity, with navy conveying authority and reliability, while gold accents introduce luxury and high quality. Careful contrast adjustments make the gold elements stand out against the dark navy, creating a striking visual contrast. The globe's curved lines were meticulously edited for perfect symmetry and balance, creating harmony in the design. For typography, the Montserrat font was chosen for its clean, modern, and readable style, with size and positioning adjusted to complement the globe design. Placing "Eclipse Publishing" below the globe reinforces structure and stability. The logo's minimalist approach ensures versatility across various applications, reflecting the publishing house's global, reliable, and creative nature. By combining thoughtful shapes, colors, and spacing, the logo effectively communicates the organization's values and mission, creating a strong and recognizable brand identity.
